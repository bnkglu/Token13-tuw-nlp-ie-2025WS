# Analysis Module

Tools for evaluating system performance and analyzing data patterns.

## Components

- **`evaluation.py`**: Functions for computing metrics (Precision/Recall/F1), generating classification reports, confusion matrices, and error analysis.
- **`mining.py`**: Utilities for analyzing training data to discover potential patterns (data-driven exploration) and visualizing rule distributions.

## Usage

```python
from src.analysis.evaluation import compute_metrics, save_all_outputs
from src.analysis.mining import visualize_rules_by_relation
```
